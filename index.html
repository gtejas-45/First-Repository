Basic commands of git

Set your username and email for Git commits.

    git config --global user.name "Your Name"
    git config --global user.email "you@example.com"

git clone <--repo_url-->

Clone an existing repository from a remote server.

It is used to to clone the repo. from github to local pc from this we can access the file from github to our pc and do changes

git status :

It is used to check what changes you do in repo. that may be creating new repo. or modifying old file

git add 
    git add .      # Add all files

git commit -m "Your commit message"

git push origin main

push the all changes to the github
